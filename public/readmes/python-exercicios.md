# Exercícios de Introdução à Programação com Python

Este é meu repositório pessoal de estudos do livro
[*Introdução à Programação com Python: algoritmos e lógica de programação para
iniciantes*, 4ª edição, de Nilo Ney Coutinho Menezes](https://www.mercadolivre.com.br/introduco-programaco-com-python-algoritmos-e-logica-de-programaco-para-iniciantes-de-nilo-ney-coutinho-menezes-editorial-novatec-edico-4-edico-em-portugus/p/MLB39584079).

Aqui registro as soluções que desenvolvi enquanto estudo. Cada exercício fica
em seu próprio arquivo, com uma demonstração no final para facilitar a
execução e a revisão do conteúdo.

## Sobre os enunciados e contextos

Os enunciados presentes nos arquivos foram colocados por inteiro. Para não
deixar as questões soltas, pedi auxílio de IA para identificar e organizar os
contextos necessários apresentados pelo livro, como classes, programas-base e
trechos que cada exercício pede para modificar.

Esses contextos também aparecem nos READMEs dos capítulos. A intenção é que
alguém consiga abrir uma solução e entender qual problema está sendo resolvido,
mesmo sem estar com a página correspondente do livro aberta.

Os textos e códigos de referência servem somente para contextualizar meu
estudo. Este repositório não substitui o livro.
