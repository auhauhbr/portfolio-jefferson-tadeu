# Créditos e atribuição

## Template original

Este repositório é uma obra derivada de:

- **Projeto:** Portfolio Template
- **Autoria:** Elomari Hana
- **Repositório:** https://github.com/ElomariHana/portfolio-template
- **Licença declarada no README original:** MIT

O template forneceu a estrutura inicial em Next.js, incluindo páginas básicas,
tema claro/escuro, navegação, animações, componentes visuais e configuração
inicial do portfólio.

## Adaptação

A versão deste repositório foi extensivamente modificada por Jefferson Tadeu dos
Santos. Entre as mudanças estão:

- identidade visual e conteúdo em português;
- dados profissionais reais;
- catálogo e filtros avançados de projetos;
- galerias, modais e READMEs integrados;
- cursos, certificados e conquistas;
- tecnologias com logos;
- currículo em PDF;
- contatos e WhatsApp;
- novos componentes e estruturas de configuração;
- deploy e domínio na Vercel.

O histórico Git original foi mantido para preservar a origem técnica do projeto.

Fotografias, currículo, certificados, textos biográficos e dados pessoais de
